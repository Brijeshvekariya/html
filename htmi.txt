HTML :- Hyper Text Markup Language
--> It is used to build a structure of HTML

website:-

It is collection of webpages which gives information

www :- World Wide Web --> Global Server

HTTP/s :- Hyper Text Transfer Protocol/secure

URL :- Uniform Resource Locator

Domain :- .com/.in/.edu/.gov/.info/.blogspot

HTML is a markup Language and not a programming Language

Markup Language :-

1. It does not show any error
2. it is not case sensitive language
3. It consists of tags and attributes

<> start tags
</> end tags

structure of HTML:

<HTML>
<HEAD>
    <TITLE></TITLE>
</HEAD>
<BODY>
    tags and attributes
</BODY>
</HTML>
